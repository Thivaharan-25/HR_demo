Connectivity Bridges — Event-Driven (MediatR)
4 Bridges • ALL MISSING

B1 — People Sync (HR → Work)

EmployeeCreated → Create WorkspaceMember + Assign Work Role + Add to Teams
EmployeeTerminated → Deactivate Work Access + Reassign Tasks + Archive + Notify
EmployeeTransferred → Update Team Memberships in Work
EmployeePromoted → Upgrade Work Role Permissions

B2 — Availability (HR → Work)

LeaveApproved → Create AvailabilityBlock + Resource Flag + AI Block + Notify Manager
LeaveCancelled → Remove AvailabilityBlock + Notify
PublicHolidayAdded → Create Org Calendar Block + Adjust Sprint Velocity
AttendanceAnomaly → Soft Flag Workload Warning

B3 — Performance (Work → HR)

TaskCompleted → Write HR PerformanceFeedEntry (task_count, on_time_rate)
SprintCompleted → Write Velocity per Member to HR + Contribution %
BudgetExceeded → Flag Project Lead in HR Performance
ProjectCompleted → Write Summary Metrics to HR Review

B4 — Skills & Talent (Bidirectional)

HR: SkillMatrixUpdated → Update Work AI Assignment Index + Smart Suggestions
Work: TaskCompleted (skill tag) → Create SkillDemonstration Record + Talent DNA + Confidence Score
HR: SkillValidated → Update "Manager Confirmed" in Work + Boosted Recommendation
